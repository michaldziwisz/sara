# 0.0.19

## Changes

- Added a jingle module: a single `.sarajingles` set with unlimited pages (10 slots per page) playable from the main window when a music playlist list is focused (`1..0`, `-`/`=`, `Shift+digit` overlay).
- Added a `Tools → Jingles…` dialog to manage pages/slots and select the output device for jingles.
