# Release 0.0.3

## English

### Highlights
- Fully integrated NVDA workflow: the app now deploys its own sleep helper, emits Play Next signals, and ships an updated NVDA add-on (0.1.33) where arrow navigation always speaks while Play Next remains silent. The GitHub build bundles the `.nvda-addon` file and attaches it to releases automatically.
- System clipboard paste: playlists can paste files and entire folders straight from the OS clipboard, turning them into items with extracted metadata.
- Playback stability: BASS and sounddevice fades no longer cancel each other, preventing the “next track sometimes stalls” issue when you trigger transitions quickly.
- Usability polish: the shortcut capture dialog handles raw Ctrl keys, shows the pressed combo in a focusable textbox, and keeps OK disabled until a binding is captured. README documents the optional Qt playlist experiment, and the project exposes the `qt` extra in `pyproject.toml`.

### Packaging
- `pyproject.toml` bumped to 0.0.3 and includes the `qt` extra plus the NVDA helper in `sara.ui`.
- The Windows workflow now runs tests, packages the NVDA add-on, and uploads both `SARA-windows.zip` and `sara-silent-addon.nvda-addon` when a tag is published.

## Polski

### Najważniejsze zmiany
- Pełna integracja z NVDA: aplikacja instaluje własny moduł „sleep”, wysyła sygnały Play Next i dołącza dodatek NVDA 0.1.33. Strzałki zawsze czytają zaznaczenie, a Play Next milczy; workflow GitHuba automatycznie dołącza plik `.nvda-addon` do releasu.
- Wklejanie ze schowka systemowego: możesz wkleić pojedyncze pliki, wiele plików lub całe foldery – zostaną dodane do playlisty z odczytanymi metadanymi.
- Stabilniejsze odtwarzanie: wygaszenia w BASS i sounddevice nie blokują już kolejnych przejść, więc szybkie przełączanie utworów nie powoduje ciszy.
- Ulepszenia UX: okno przechwytywania skrótów rozpoznaje surowy Ctrl, wyświetla kombinację w polu tekstowym i aktywuje OK dopiero po złapaniu skrótu. README opisuje eksperymentalną playlistę Qt, a `pyproject.toml` ma teraz opcję `qt`.

### Pakowanie
- `pyproject.toml` podbity do 0.0.3 z nową opcją `qt`, a moduł NVDA (`sara.ui.nvda_sleep`) wchodzi do paczki.
- Workflow Windows uruchamia testy, pakuje dodatek NVDA i dołącza zarówno `SARA-windows.zip`, jak i `sara-silent-addon.nvda-addon` do wydania.
