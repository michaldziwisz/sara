# 0.0.13

## Nowości
- Dodano nowy typ playlisty „Folder muzyczny”: wskazujesz katalog z plikami audio, lista ładuje się jako osobny panel, `Spacja` odpala odsłuch na urządzeniu PFL, a `Enter` wrzuca zaznaczone utwory do ostatnio używanej playlisty muzycznej. Folder można zmieniać z paska nad listą lub z opcji startowych.
- `Ctrl+P` (konfigurowalne) otwiera dialog punktów miksu dla bieżącego utworu, także w playlistach folderowych; kontekstowe menu również udostępnia tę funkcję.
- Wpisy startowe w oknie opcji obsługują teraz zarówno klasyczne playlisty muzyczne/newsowe, jak i foldery, włącznie z wyborem katalogu.

## Poprawki
- Naprawiono obsługę przycisku „Wybierz folder…” w opcjach – dialog przestaje zgłaszać wyjątki.
- Uzupełniono brakujące polskie tłumaczenia w oknie modyfikacji punktów miksu.
- Release tooling aktualizuje teraz wersję 0.0.13 i pakuje te same artefakty co poprzednie wydanie.

## Znane uwagi
- Budowanie wxPython lokalnie nadal wymaga GTK (Linux) lub Visual Studio Build Tools (Windows).
- Pełne wsparcie ASIO/WASAPI/Qt pozostaje w extrasach: `pip install -e ".[dev,asio,wasapi,qt]"`.
