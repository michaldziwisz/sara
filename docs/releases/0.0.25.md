# 0.0.25

## Changes

- Feature: add playlist item types (`song` / `spot`) with UI markers and configurable shortcuts for quick tagging.
- Feature: add a `Logging` options tab with played-tracks CSV logging (hourly files) and per-type filters.
- Feature: add `nowplaying.txt` support with configurable template, update triggers, output location, and per-type filters.
- Fix: restore Polish characters in audio device names (BASS/ASIO decoding regression after refactor).
