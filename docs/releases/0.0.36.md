# 0.0.36

## Changes

- Perf: add `thread` mix trigger executor to reduce automix timing jitter (config: `playback.mix_executor` in Options / `settings.yaml`).
- Perf: improve BASS fade-out by using native volume slide instead of per-step volume updates.
- Perf: make audio preloading effective across slots by caching players per `(playlist, slot)` so the next start reuses the preloaded stream.
- Diagnostics: log `MIX_METRIC` for native trigger callback-to-execution delay measurements.
