# Release 0.0.4

## English

### Highlights
- **News playlists revamped.** Markdown editing now supports headings up to `#####`, audio placeholders paste directly from the OS clipboard, read mode renders wrapped text in a single control, and keyboard shortcuts (`H`/`Shift+H`, `C`/`Shift+C`, `Enter`/`Space`) let you navigate and play clips without leaving the list.
- **Portable news services.** Playlists can import/export `.saranews` JSON packages (buttons or `Ctrl+O`/`Ctrl+S`), and the new `sara-news-editor` standalone app reuses the same UI so newsroom staff can prepare scripts outside the main automation.
- **Flexible playback devices.** Both SARA and the standalone editor let you choose an output card for news clips. The editor remembers the last device in `config/news_editor.yaml`, making it easy to preview services on shared workstations.
- **Configurable news auto-start.** Options → Startup playlists now lets you add news playlists (not only music) so SARA can open newsroom panels automatically at launch.
- **Accessibility polish.** The NVDA add-on and app-side hooks ensure arrow navigation always speaks every line, Play Next stays silent, and news menus/dialogues keep talking even when playback shortcuts are triggered quickly.

### Packaging
- `pyproject.toml` bumped to 0.0.4 and exposes an extra console entry (`sara-news-editor`) for the standalone tool.
- New configuration file `config/news_editor.yaml` (auto-generated) keeps the editor’s device choice separate from the main SARA settings.

## Polski

### Najważniejsze zmiany
- **Odświeżone playlisty newsowe.** Edycja Markdown obsługuje nagłówki do `#####`, dźwięki można wklejać bezpośrednio ze schowka, tryb odczytu pokazuje całość w jednym polu, a skróty (`H`/`Shift+H`, `C`/`Shift+C`, `Enter`/`Spacja`) pozwalają nawigować i uruchamiać klipy bez opuszczania listy.
- **Przenośne serwisy.** Playlisty potrafią importować/eksportować paczki `.saranews` (`Ctrl+O`/`Ctrl+S`), a nowa aplikacja `sara-news-editor` wykorzystuje ten sam interfejs, więc redakcja może przygotować serwis poza główną Sarą.
- **Elastyczne urządzenia audio.** W Sarze i w edytorze można wskazać kartę dźwiękową dla klipów newsowych; edytor zapamiętuje ostatni wybór w `config/news_editor.yaml`, co ułatwia odsłuch na współdzielonych stanowiskach.
- **Newsowe playlisty przy starcie.** W opcjach (Startup playlists) można teraz dodawać także playlisty newsowe, dzięki czemu po uruchomieniu Sary automatycznie otwierają się zarówno listy muzyczne, jak i newsroomowe.
- **Dostępność.** Dodatek NVDA wraz z poprawkami po stronie aplikacji gwarantuje, że strzałki zawsze czytają każdy wiersz, Play Next milczy, a menu i okna newsowe nie tracą mowy nawet przy szybkim przełączaniu.

### Pakowanie
- `pyproject.toml` podbity do 0.0.4 z nowym skrótem `sara-news-editor`.
- Niezależny plik konfiguracyjny `config/news_editor.yaml` (tworzony automatycznie) przechowuje wybór urządzenia edytora newsów i nie miesza się z główną konfiguracją Sary.
