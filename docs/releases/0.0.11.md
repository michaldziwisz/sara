# 0.0.11

## Nowości
- Refaktoryzacja panelu newsowego: logika trybu edycji/czytania przeniesiona do kontrolerów (`NewsEditController`, `NewsReadController`), delegowane skróty wklejania/odsłuchu i nawigacji markerów/nagłówków.
- Nowe serwisy UI: zarządzanie odtwarzaniem/preview, layoutem playlist, schowkiem i undo jako osobne klasy z testami jednostkowymi.
- Usprawniony wybór plików: `FileSelectionDialog` korzysta z `FileBrowser`, lepsze zapamiętywanie ścieżek i obsługa wzorców.
- Wzmocniona obsługa schowka na Windows (audio, Drag&Drop) – zabezpieczenia przed błędnymi uchwytami/AV, komunikaty spójne z panelem.

## Poprawki
- Naprawiony skrót Tab w trybie czytania (brakujące `keycode`).
- Callbacki odtwarzania zabezpieczone na wypadek braku `wx.App` (defensywnie w środowiskach testowych).
- Usunięty stary test panelu newsowego, nowe testy dla kontrolerów, schowka, layoutu playlist i serwisów.

## Znane uwagi
- Kompilacja wxPython w środowisku deweloperskim wymaga bibliotek GTK (Linux) lub Visual Studio Build Tools (Windows).
- Pełne pakiety ASIO/WASAPI/Qt nadal opcjonalne – instaluj przez extras `pip install -e ".[dev,asio,wasapi,qt]"`.
