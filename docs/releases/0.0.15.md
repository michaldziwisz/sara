# 0.0.15

## Zmiany i poprawki
- `Play Next` w automixie pomija teraz elementy z aktywnym `loop_hold`. Po wciśnięciu Next pętla jest łagodnie zatrzymywana, stan auto-mix aktualizuje kursor i natychmiast startuje następny wpis z playlisty, więc podkład nie restartuje się od początku.
- Scenariusz został dodany do zestawu testów (`test_loop_hold_play_next_advances_cursor`), aby regresja nie wróciła w kolejnych wydaniach.
